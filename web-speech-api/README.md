# Web Speech API Color Changer
This has been adapted from [MDN's Speech Color Changer example.](https://github.com/mdn/web-speech-api/tree/master/speech-color-changer)

## Part 1:
1. `nvm use`
2. `npm i`
3. `npm start`
4. `http://localhost:5000/`
5. Say a color.

## Part 2:
Try changing out colors for something else that a user could say.
